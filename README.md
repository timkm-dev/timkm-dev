# Hi there, I'm Timothy Kimani 👋

I'm a Computer Science student passionate about building intelligent systems that work in production, from raw data pipelines to deployed models. I enjoy the full arc of ML engineering: designing robust ETL infrastructure, training and evaluating models, and wiring everything into end-to-end systems that solve real problems. Lately I've been drawn to anomaly detection, time-series modeling, and the infrastructure that makes ML reliable at scale.

- 🔭 **Current Focus:** Building a predictive maintenance system for hydroelectric power plants — end-to-end, from sensor data ingestion and ETL pipelines to LSTM-Autoencoder anomaly detection models in production
- 🌱 **Learning & Exploration:** Deep learning architectures (LSTMs, Autoencoders), probabilistic modeling, distributed data processing with Spark, and C++ systems programming
- ⚡ **Core Interests:** ML Engineering, Time-Series & Anomaly Detection, Data Pipeline Architecture, End-to-End AI Systems, Embedded & IoT Systems
